Android项目之公益活动app
=======
项目介绍：
-------
### 功能介绍
* 1.用户注册、登录、找回密码
* 2.近期公益活动的信息、活动的报名、活动发布
* 3.活动的列表以及活动详细信息
* 4.客服机器人
### 运用知识
* 数据库的基本查询操作
* 后端数据库使用的的是第三方的Bmob云，所以要根据其提供的文档进行数据库的操作
* 使用Fragment来实现点击下方导航框切换页面
* 自动聊天机器人使用的是图灵的机器人，可以实现自动的回复
### 准备
* Bmob的数据库SDK
* 图灵机器人的SDK
* xUtils-2.6.14.jar
* android-support-v4.jar
### 开发环境
* Android studio
